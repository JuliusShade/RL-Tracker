# Rank Badge Assets

This directory should contain rank badge images for Rocket League ranks.

## Required Images

You can download rank badges from Rocket League Tracker or create your own:

- `unranked.png`
- `bronze.png`
- `silver.png`
- `gold.png`
- `platinum.png`
- `diamond.png`
- `champion.png`
- `grand-champion.png`
- `supersonic-legend.png`

## Image Specifications

- Format: PNG with transparency
- Recommended size: 128x128 pixels or larger
- Keep aspect ratio square for best display

## Sources

You can find official Rocket League rank icons at:
- https://rocketleague.tracker.network (right-click on rank badges)
- Official Rocket League assets
- Community-created rank badge sets

## Usage

The application will automatically load these images when displaying rank information.
If an image is not found, the app will still function but won't display the rank badge.
